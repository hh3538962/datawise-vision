# DataWise Vision

AI Powered E-Commerce Analytics Platform for Machine Learning Business Analytics

## Live Project

🌐 Domain: `datawise-vision.lovable.app`

---

# Overview

DataWise Vision is a modern AI powered Business Analytics platform developed for the Machine Learning for Business Analytics course project.

The platform combines:

* Machine Learning
* Business Intelligence
* Data Visualization
* Predictive Analytics
* Interactive Dashboards
* E-Commerce Insights

The system performs both:

* Classification Analysis
* Regression Analysis

using multiple machine learning algorithms and presents results through a professional web interface.
